Infrastructure Monitoring

Big brother is watching your infrastructure. Here's what he sees:

  • Uchiwa: Dashboard for Sensu, the open-source monitoring framework
  • Sensu API: The Sensu API, useful if you'd like to pull Sensu data into something else
  • Graphite: Real time graphing; Sensu can ship metrics to Graphite
  • Graphite Dashboard: You can build dashboards for specific use cases (e.g. key stats for production web servers)
  • Grafana: A richer/more modern dashboard for Graphite data
